### 功能

> [!Important] 
> AI功能需自行向 AI运营商 付费获得 API（你也可以自己寻找免费途径）
- [x] 原生支持 Gemini、ChatGPT、DeepSeek 模型
- [x] 支持 上下文、预设 保存
- [x] Jianer插件兼容
- [x] 简儿有的都该有吧
> [!Tip]
> ……您可以在部署完成后，@您的机器人查看功能，或者发送“{您设置的触发前缀}帮助”（无需带上{}）来查看管理权限
